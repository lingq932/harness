# 个人求职站点 — Spec

## 目标

搭建一个面向 HR 的 AI 产品经理个人站点。核心用途：求职打招呼阶段，HR 点开链接即可快速了解过往经历和个人亮点。本质是"可交互的超级简历"，不是内容博客。

## 范围

### 页面结构

1. **首页**
   - 个人职业照（圆角大图，给足展示空间）
   - 一句话定位（例：AI 产品经理 / X 年经验）
   - 核心亮点标签（3-5 个关键词）
   - 导航入口引导 HR 继续浏览
   - 不堆文字，第一眼抓住"这个人是做什么的"

2. **工作经历**
   - 时间线形式展示
   - 突出 AI / 产品相关成果
   - 内容来源：现有简历

3. **个人作品**
   - 卡片式布局
   - 每张卡片支持不同形态的链接跳转（外部网页、PDF、HTML 等）
   - 初期先放占位卡片，后续逐个填入真实内容和链接

4. **联系方式**
   - 微信 / 邮箱 / LinkedIn
   - 底部固定区块或独立区块

### 视觉风格

- 高级简约、有设计感
- 主色调：白底 + 黑色文字，大量留白，参考 Katerina 风格个站
- 超大号名字作为视觉主焦点，左文右图布局（PC 端）
- 顶部导航栏：左名字 / 中导航 / 右联系按钮
- 响应式适配：PC 端和移动端均可正常浏览，布局自动适应屏幕宽度

## 技术方案

- 框架：Next.js
- 样式：Tailwind CSS（内置响应式断点，移动端优先策略）
- 部署：Vercel（SSG 静态导出，零成本上线）
- 语言：仅中文

## 约束

- 首页文字内容从简，不做大段自我介绍
- 个人作品区初期仅占位，不实现作品内容本身
- 所有内容数据硬编码或 Markdown 文件管理，不接数据库

## 暂不处理项

- 英文版 / 多语言
- 作品区真实内容填充
- 暗色模式
- SEO 优化
- 访问数据统计
- 博客 / 文章功能
